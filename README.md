# Appel-D-offre
Conception et développement d'une plateforme web intelligente de gestion des appels d'offres avec IA : pipeline RAG pour l'analyse documentaire, scoring multicritère automatisé avec agents LangChain, et observabilité via Langfuse. (React, FastAPI, LangChain, LiteLLM, ChromaDB, PostgreSQL, Docker Compose)
